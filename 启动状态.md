## 前端服务 ⚠️ 需要手动启动

由于系统未安装Node.js，前端服务需要手动安装和启动。

### 安装步骤

1. **安装Node.js** (建议版本 16+)
   ```bash
   # macOS 使用 Homebrew
   brew install node
   
   # 或访问 https://nodejs.org 下载安装
   ```

2. **安装前端依赖**
   ```bash
   cd /Users/zhengsonghao/CodeBuddy/20260317162027/frontend
   npm install
   ```

3. **启动前端开发服务器**
   ```bash
   npm run dev
   ```

4. **访问前端界面**
   - 打开浏览器访问: http://localhost:5173
   - 前端会自动代理API请求到后端 http://localhost:5002
