## 具体步骤

### 第一步：部署前端到Vercel

**时间：2-3分钟**

1. 访问 https://vercel.com/signup
2. 点击 "Continue with GitHub"
3. 授权Vercel访问GitHub
4. 点击 "Add New..." → "Project"
5. 选择仓库：`shzheng022-prog/quant-platform`
6. 配置：
   - **Framework Preset**: Vite
   - **Root Directory**: `frontend`
   - **Build Command**: `npm install && npm run build`
   - **Output Directory**: `dist`
7. 点击 "Deploy"
8. 等待1-2分钟，部署完成！

**获得前端URL**：
```
https://quant-platform.vercel.app
```

### 第二步：部署后端到Glitch

**时间：5-10分钟**

#### 选项A：使用Glitch（推荐）

1. 访问 https://glitch.com
2. 点击 "New Project" → "glitch-hello-python"
3. 修改代码：

将Glitch项目中的 `server.py` 替换为您的Flask应用：

```python
from flask import Flask
from flask_cors import CORS
import sys
import os

# 复制您的app.py内容到这里
# 从当前项目复制 backend/app.py 的内容

app = Flask(__name__)
CORS(app)

# 添加所有的路由和函数...

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=3000)
```

4. 添加依赖：
   - 在项目中创建 `requirements.txt`
   - 添加：`Flask Flask-CORS pandas numpy akshare scipy`

5. Glitch会自动安装依赖和部署

6. 获得后端URL，如：
   ```
   https://quant-platform-backend.glitch.me
   ```

#### 选项B：使用Replit

1. 访问 https://replit.com
2. 点击 "Create Repl"
3. 选择 "Python" → "Flask App"
4. 将您的 `backend/app.py` 内容复制到 `main.py`
5. 点击 "Run" 按钮
6. 获得后端URL

### 第三步：连接前后端

获取后端URL后，配置前端：

**方法1：在Vercel添加环境变量**

1. 在Vercel项目页面，点击 "Settings" → "Environment Variables"
2. 添加变量：
   ```
   Name: VITE_API_BASE_URL
   Value: https://quant-platform-backend.glitch.me
   ```
3. 重新部署前端

**方法2：直接修改代码**

编辑 `frontend/src/api.js`：

```javascript
const API_BASE_URL = 'https://quant-platform-backend.glitch.me'
```

然后重新部署前端。
