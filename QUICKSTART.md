# 快速使用指南

## 项目已完成 ✅

你的量化策略回测网站已经搭建完成！这是一个全功能的Web平台，可以完美运行你的聚宽"七星高照ETF轮动策略"。

## 立即开始

### 方式一：手动启动（推荐）

#### 1. 启动后端服务

```bash
# 安装Python依赖
pip install -r requirements.txt

# 启动后端
cd backend
python app.py
```

后端将运行在 `http://localhost:5000`

#### 2. 启动前端服务

打开新的终端窗口：

```bash
# 安装前端依赖
cd frontend
npm install

# 启动前端
npm run dev
```

前端将运行在 `http://localhost:5173`

#### 3. 访问平台

在浏览器中打开：http://localhost:5173

### 方式二：使用启动脚本

```bash
# 给脚本添加执行权限
chmod +x start.sh

# 运行启动脚本
./start.sh
```

## 常见问题

### Q: 后端启动失败？
A: 检查是否安装了所有依赖：
```bash
pip install Flask Flask-CORS akshare pandas numpy scipy python-dateutil pytz
```

### Q: 前端启动失败？
A: 检查是否安装了Node.js和npm：
```bash
node -v
npm -v
```

### Q: 获取不到实时数据？
A: AkShare接口可能有限制，稍后再试或检查网络连接。

### Q: 回测结果不准确？
A: 本平台为简化版回测，主要用于验证策略逻辑，不作为投资依据。
